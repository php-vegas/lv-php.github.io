Overview

Welcome to the GitHub location of the Las Vegas PHP Users Group. We store our projects an code examples here. All repositories are public, are member contributed, and considered open source> Licensing is project based. Please refer to the license details in each project.

Group

If you would like to join our users group, please see our meet up page at http://www.meetup.com/Las-Vegas-PHP-Users-Group/

Projects

We are currently working on implementing the security API for local Las Vegas start up LaunchKey into popular PHP frameworks as releasing them as open source. If you would like to contribute to the effort, you can join one of the projects. If you would like to suggest other open source projects for our group, please submit them to one of the organizers listed below.

Authors and Contributors

The group is currently organized by Adam Englander (@aenglander) and Josh Copeland (@psycodedotorg). Individual projects are managed by project leads. Please see the individual project pages for more information regarding project leaders.
